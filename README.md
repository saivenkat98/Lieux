# Lieux

✨ What is ‘Lieux’?
<br/>'Lieux' is a minimalist social platform (built using MERN stack) that allows users to share their favorite places 🌆🏞️, engage with others' choices, discover new locations . Each user can:
<br/> ✅ Add, edit, and delete places 📝
<br/> ✅ Attach live locations for real-time updates 📍
<br/> ✅ Explore places loved by others and find new favorites 🚀
